风场：同发、镇赉，品牌：华锐，机型：SL1500 ABB PLC机组，故障代码：277，故障解释：变频器通信复位错误，故障类型：变频器系统，故障原因：PLC与变频器CAN通讯发生故障，故障现象及处理方法：1.检查变频器至 PLC 的 CAN 通讯接口之间线路、器件是否松动。2.更换主 PLC。
风场：同发、镇赉，品牌：华锐，机型：SL1500 ABB PLC机组，故障代码：278，故障解释：变频器的多路转换器通讯错误，故障类型：变频器系统，故障原因：状态机不在 PLC 停止状态、PLC 初始化状态、PLC 自检状态或变频器复位状态时,变频器与主 PLC 之前通讯中断。，故障现象及处理方法：1.检查变频器至 CAN-spider 之间的光纤连接是否正常,CAN-spider(NCC310-A293.2)上通讯光纤正常有一个口有光。2.检查 CAN-spider 至 PLC 之间的 CAN 通讯线及终端电阻是否正常。3.检查变频器上电源板指示灯是否正常、变频器供电电源是否正常,如NCC310-F352.4 熔丝、网侧接触器 K340.4 辅助触点是否正常。4.检查主站 PLC 的 CAN-Open 协议是否正常,下载 PLC 的 CAN 协议时需要 PLC处于 Stop 模式(可以通过 AC1131 登入后点击 Stop 或将 PLC 右上方的拨码拨到 Stop 位置)。
风场：同发、镇赉，品牌：华锐，机型：SL1500 ABB PLC机组，故障代码：279，故障解释：变桨柜1通讯错误，故障类型：变桨系统，故障原因：变桨1变频器与 PLC 间通讯中断，故障现象及处理方法：1.如有变桨子故障,参考变桨故障处理手册。2.如果Lenze系统做Crowbar 测试报,检查Crowbar 测试时,24V 电源是否下降很大(如:降到 13V 左右),导致变桨变频器电源瞬间供电不足。3.如果一偏航就报,检查偏航启动时,变桨通讯线缆 CAN-H 与 CAN-L 间的电阻跳变,正常为120欧姆。如果跳变,分段检查问题产生点。4.检查轮毂内控制柜之间的电缆是否固定良好(扎带固定好,不会来回甩动)。5.检查PLC的CAN通讯接口到偏航变频器CAN口、变桨1变频器CAN之间的线路是否松动。6.检查偏航变频器、变桨变频器CAN插头上的屏蔽线是否连接正常。7.检查变桨通讯回路浪涌。
