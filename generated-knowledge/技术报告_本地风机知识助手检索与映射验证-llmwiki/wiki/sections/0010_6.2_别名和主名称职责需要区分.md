# 6.2 别名和主名称职责需要区分

来源：技术报告_本地风机知识助手检索与映射验证.md

当前部分用户常用说法被放在了 `site` 主字段中。更合理的结构应为：

- `site`：标准文件中的规范场站名，用于最终答案展示。
- `aliases`：用户可能输入的别名、长名称、俗称，用于匹配。

例如当前结构：

```json
{
  "site": "新华风电场",
  "aliases": ["新华"],
  "models": ["三一 SE8715", "华仪 HW2/S1500(87)", "运达 WD88-1500A"]
}
```

建议调整为：

```json
{
  "site": "新华",
  "aliases": ["新华风电场"],
  "models": ["三一 SE8715", "华仪 HW2/S1500(87)", "运达 WD88-1500A"]
}
```

这样用户输入“新华风电场”仍然可以命中，但系统输出会保持标准名称“新华”。
